# Truss Local Providers

<p align="center"><img src="./logo.png" width="96" alt="Truss logo"></p>

This package provides streaming local-model adapters for native Ollama and OpenAI-compatible servers such as LM Studio and llama.cpp. It is used by the Truss CLI, TUI, and VS Code extension.

For end-user setup, use `@truss-harness/cli` or `@truss-harness/tui`.
